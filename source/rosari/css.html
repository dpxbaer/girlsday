CSS

Was ist CSS?

CSS = Cascading Style Sheet

Aussehen der HTML-Elemente festlegen

Syntax

  • Inline: mit einem Style-Attribut <p style="color:red";>
  • Intern: mit einem <style> - Element
  • Extern: den CSS-File mit in dem <head> - Tag des HTML einbinden

Selektoren und Deklarationen

Selektor

  • Type: Typ des HTML ELements
  • Class: Klassen werden verwendet, um die gleichen Styles für mehrere Elemente anzuwenden
  • ID: Styles für ein einziges HTML-Element; kann nur einmal verwendet werden
  • Pseudoklasse: Styles für bestimmte Zustände der Elemente, z.B. hover oder focus

Deklaration

  • Eigenschaft (property): welche visuelle Merkmale sollen verändert werden
  • Wert (value): wie die Eigenschaft verändert werden soll

Reihenfolge

  • In einem HTML können mehrere Stylesheets verwendet werden
  • Das HTML muss wissen, welche Stylesheet-Regeln für welche HTML-Elemente gelten
    1. ID
    2. Class
    3. ID
